# 📋 Mini CRM – Client Lead Management System

A simple and responsive Client Lead Management System developed as part of the Future Interns Full Stack Development Internship.

## 🚀 Features

- ➕ Add New Lead
- ✏️ Edit Existing Lead
- 🗑️ Delete Lead
- 🔍 Search Leads
- 📂 Filter by Status
- 📊 Dashboard Statistics
- 💾 Local Storage Support
- 📱 Responsive User Interface

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Version Control
- Git
- GitHub

## 🎯 Project Objective

The objective of this project is to manage customer leads efficiently by allowing users to add, edit, delete, search, and organize lead information through an intuitive interface.
